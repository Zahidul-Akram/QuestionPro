## Description

This project provides an API that returns employee information (hierarchy by position) under any given position in the organizational structure. It features JWT token authentication and uses **NestJS** as the framework.

The API is designed for efficient querying of employee data, and it is optimized for load testing with **Artillery** to ensure it can handle high traffic.

## Technologies Used

- **NestJS**: Framework for building efficient server-side applications.
- **PostgreSQL**: RDBMS used to store employee data.
- **TypeORM**: ORM for interacting with PostgreSQL.
- **JWT Authentication**: For securing API endpoints.
- **Artillery**: Load testing tool to simulate high traffic.
